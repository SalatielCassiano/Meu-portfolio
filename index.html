Sistema de cadastro de games - Python

O código é um programa que permite aos usuários cadastrar, listar e excluir jogos de videogame em um arquivo de texto. O arquivo "games.txt" é utilizado para armazenar os dados dos jogos e seus respectivos videogames. Se o arquivo não existir no início, o programa o criará automaticamente. O código visa fornecer uma funcionalidade básica de cadastro e gerenciamento de jogos de videogame por meio de um menu simples e interativo.
